# Task Management App

## About
Task Management App is a web-based system for organizing coursework and assignments. It supports role-based experiences for students, lecturers, and admins, with dashboards and task workflows tailored to each role.

## Features
- Role-based access (student, lecturer, admin)
- Task creation, editing, and completion tracking
- List and Kanban views for tasks
- Search and status filtering
- Courses, calendar, notifications, and dashboards

## Tech Stack
### Frontend
- Next.js 14 + React 19
- TypeScript
- Tailwind CSS + Radix UI components
- React Hook Form + Zod validation

### Backend
- Node.js + Express
- Prisma ORM with MySQL
- TypeScript

### Tooling & Testing
- pnpm workspaces
- Vitest + Supertest
- ts-node-dev for backend development

## Scripts
Run from the repo root:
- `pnpm dev` - Start frontend and backend in dev mode
- `pnpm dev:frontend` - Start only the frontend
- `pnpm dev:backend` - Start only the backend
- `pnpm build` - Build both apps
- `pnpm start` - Start both apps
- `pnpm test` - Run tests
